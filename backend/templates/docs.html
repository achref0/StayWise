API Documentation

City Search

GET /api/city_search
Parameters:
- cityid (required): City ID
- checkin (required): Check-in date (YYYY-MM-DD)
- checkout (required): Check-out date (YYYY-MM-DD)
- adults (optional): Number of adults (default: 2)
- rooms (optional): Number of rooms (default: 1)
- cur (optional): Currency (default: USD)

Hotel Search

GET /api/hotel_search
Parameters:
- hotelid (required): Hotel ID
- checkin (required): Check-in date (YYYY-MM-DD)
- checkout (required): Check-out date (YYYY-MM-DD)
- adults (optional): Number of adults (default: 1)
- rooms (optional): Number of rooms (default: 1)

Booking.com Search

GET /api/booking_search
Parameters:
- country (required): Country code
- hotelid (required): Hotel ID
- checkin (required): Check-in date (YYYY-MM-DD)
- checkout (required): Check-out date (YYYY-MM-DD)
- adults (optional): Number of adults (default: 2)
- rooms (optional): Number of rooms (default: 1)
- currency (optional): Currency (default: USD)
- kids (optional): Number of kids (default: 0)

Mapping Search

GET /api/mapping
Parameters:
- name (required): Hotel or city name

Account Information

GET /api/account
No parameters required
